# 여자친구 음식 자랑 대회 ♥

핑크 Y2K 픽셀 감성의 요리 기록 사이트입니다.

## 기능
- 이메일 + 비밀번호만으로 회원가입/로그인
- 요리 기록: 사진 여러 장 첨부, 날짜(수정 가능), 요리 설명, 별점(1~5)
- 최상단 BEST 3 — 관리자(savntew@gmail.com)가 직접 지정/해제
- 기록 리스트 스크롤, 기록 클릭 시 상세 보기 / 수정 / 삭제

## 데이터 저장
서버 없이 브라우저(localStorage + IndexedDB)에 저장됩니다.
같은 기기·같은 브라우저에서 기록이 유지됩니다.

## 실행
정적 사이트라서 `index.html`을 열거나 아무 정적 서버로 서빙하면 됩니다.
